About these modules

Each file is a self-contained single-file HTML lesson — no external scripts, no external stylesheets, no CDN dependencies. They embed cleanly in Canvas (and other LMS platforms), open directly in any modern browser, and work offline once downloaded.

All modules support light and dark mode automatically, honor prefers-reduced-motion, meet WCAG 2.1 AA contrast and tap-target targets, and use a CSP-safe scripting pattern (no inline event handlers) so they survive strict Content Security Policies in LMS iframes.
